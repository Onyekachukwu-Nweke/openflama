# OpenFlama Web

React + TypeScript frontend for the OpenFlama forecasting engine.

## Pages

- `/` - Landing page with product showcase
- `/dashboard` - Prediction form, recent predictions, and available models

## Stack

- React 18 with TypeScript
- Vite for bundling
- TanStack Query for data fetching
- Tailwind CSS with shadcn/ui components
- Wouter for client-side routing

## Theme

Dark-first design with black background and red primary accents.

- Font: Inter (sans), JetBrains Mono (code)
- Primary: `hsl(0, 84%, 50%)` (red)
- All CSS variables support dark mode

## Running Locally

```bash
npm install
npm run dev
```
