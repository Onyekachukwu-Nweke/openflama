# Roadmap

## v0.1.0 - MVP (Current)

- [x] Forecasting engine with 4 baseline models
- [x] REST API for predictions and backtesting
- [x] Walk-forward backtesting with metrics
- [x] PostgreSQL persistence
- [x] Web dashboard with dark theme
- [x] TypeScript SDK scaffold

## v0.2.0 - Data & Models

- [ ] Real market data integration (Yahoo Finance, Alpha Vantage)
- [ ] ARIMA / exponential smoothing models
- [ ] Model comparison endpoint
- [ ] Historical data caching layer
- [ ] Python core library with model implementations

## v0.3.0 - Agents Framework

- [ ] Agent protocol specification
- [ ] First agent: Momentum Scanner
- [ ] Agent orchestration runtime
- [ ] Agent configuration schema
- [ ] Inter-agent messaging

## v0.4.0 - Production Readiness

- [ ] Authentication and API keys
- [ ] Rate limiting
- [ ] Async backtest execution with job queue
- [ ] Webhook notifications
- [ ] Monitoring and alerting

## v0.5.0 - Advanced Features

- [ ] Portfolio-level forecasting
- [ ] Custom model upload (ONNX / PMML)
- [ ] Ensemble model combiner
- [ ] Real-time streaming predictions
- [ ] Multi-asset correlation analysis
